# 🎮 Portal Gamer

Bem-vindo ao **Portal Gamer**, um site dedicado a trazer dicas, tutoriais e novidades sobre os principais jogos da atualidade: **Warzone, Fortnite e GTA V**.

## 🚀 Funcionalidades
- Menu de navegação fixo para fácil acesso às páginas.
- Páginas dedicadas para cada jogo com tutoriais e vídeos incorporados.
- Galeria de vídeos organizada em grid.
- Página de contato com formulário para envio de mensagens.
- Layout moderno com CSS customizado.

## 🛠️ Tecnologias utilizadas
- **HTML5** para estrutura das páginas.
- **CSS3** para estilização e responsividade.
- **JavaScript** (futuro) para interatividade.
- Hospedagem via **GitHub Pages**.
